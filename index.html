antkorp

Antkorp is next generation communication and colloboration platform built on foundations of HTML5 and linux. Antkorp is both a product and platform at the same time. You can extend or replace parts of antkorp with your own. Antkorp client seemlessly runs across a range of devices. Antkorp server scales horizontally across commodity linux computers to adopt the increasing demand.

What sets antkorp apart

Antkorp is not a conventional web application, it is written using many HTML5 features which are not yet standardized. Antkorp uses a custom framing on top of websockets to communicate between client and server. Using websocket helps decreases the delay and bandwidth required to accomplish the same task if it were to use HTTP. Antkorp provides video and audio communication using webRTC. No plugins and software need to be installed on the client device to communicate with other team members using antkorp. Antkorp is built as a single webpage application which is insanely responsive.
