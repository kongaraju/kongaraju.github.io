## 🚀 Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- HTTPS for PWA features (or localhost for development)

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/kongaraju/kongaraju.github.io.git
   cd kongaraju.github.io
   ```

2. **Start a local server**
   ```bash
   # Using Python
   python -m http.server 8000

   # Using Node.js
   npx http-server

   # Using PHP
   php -S localhost:8000
   ```

3. **Open in browser**
   ```
   http://localhost:8000
   ```

### Testing PWA Features

1. **Service Worker**
   - Open Chrome DevTools → Application → Service Workers
   - Verify registration

2. **Offline Mode**
   - DevTools → Network → Check "Offline"
   - Reload page - should still work

3. **Install App**
   - Look for install button (bottom-right)
   - Click to install
   - Check home screen for app icon
